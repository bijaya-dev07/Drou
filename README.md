**Drou** is a high-performance, visually stunning e-commerce landing page built with the latest web technologies. It features a minimalist aesthetic coupled with powerful animations and smooth interactions to provide a premium user experience.

---

## 🚀 Key Features

- **⚡ Modern Tech Stack**: Built with Next.js 15+ (App Router) and React 19.
- **✨ Premium Animations**: Professional-grade motion design using **GSAP** (GreenSock Animation Platform).
- **🎢 Smooth Scrolling**: Integrated with **Lenis** for a buttery-smooth desktop and mobile scrolling experience.
- **🎨 Tailwind CSS v4**: Leveraging the newest features of Tailwind for lightning-fast styling and a modern design system.
- **📱 Fully Responsive**: Optimized for all screen sizes, from mobile phones to high-resolution desktops.
- **🏷️ Product Showcases**: Dynamic grids for "Latest" and "Popular" products.
- **📰 Blog Integration**: A sleek blog section to engage users with stories and news.

## 📖 Topics You Can Learn From This Project

This codebase serves as an excellent reference for modern frontend engineering. By exploring this project, you can learn:

1.  **Next.js App Router Architecture**: How to structure a modern application using the `app/` directory, server vs. client components, and optimized layout systems.
2.  **Advanced GSAP Animations**:
    *   Creating complex timelines (`gsap.timeline()`).
    *   Implementing staggering effects for entrance animations.
    *   Handling refined easing (`power3.out`, `power4.out`) for natural motion.
3.  **Smooth Scroll Implementation**: How to wrap an entire application in a Lenis provider and synchronize it with the browser's Request Animation Frame (RAF).
4.  **Modern CSS with Tailwind v4**: Understanding the new configuration-less approach and advanced container utilities.
5.  **Component Modularity**: How to build reusable UI components like `ProductGrid`, `Hero`, and `Navbar` that are both functional and visually consistent.
6.  **Declarative UI Design**: Managing complex visual states purely through React's component-based paradigm.

## 📥 Getting Started

### Prerequisites

- **Node.js**: v18.x or higher
- **npm / yarn / pnpm**

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/drou.git
    cd drou
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    # or
    yarn install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```

4.  **Open the app:**
    Navigate to [http://localhost:3000](http://localhost:3000) in your browser.
